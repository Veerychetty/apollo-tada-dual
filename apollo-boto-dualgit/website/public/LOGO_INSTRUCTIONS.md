# Logo Setup Instructions

## Adding the ESSENSE LABS Logo

1. Place your ESSENSE LABS logo image in this `public` folder
2. Name it: `essense-labs-logo.png`
3. The logo should be in PNG format with a transparent or black background
4. Recommended dimensions: width ~300-400px, height ~60-80px (or maintain aspect ratio)

The header will automatically display the logo. If the image is not found, it will fallback to text.
